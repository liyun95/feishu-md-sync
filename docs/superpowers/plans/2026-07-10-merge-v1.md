# Merge V1 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a Git-like `feishu-md-sync merge` command that merges remote Feishu Markdown changes back into the local source Markdown safely.

**Architecture:** Keep `merge` as a local working-tree operation: it may fetch Feishu Markdown through the existing `FeishuAdapter`, but it never writes Feishu, publish receipts, or base snapshots. Add publish-time local base snapshots so future merges can do three-way line merges. Use an internal line-based merge engine with conflict markers, plus abort state for restoring the local file after an in-place merge.

**Tech Stack:** Node 20+, TypeScript ESM, Commander, Vitest, existing `lark-cli` adapter.

---

## Decisions Locked

- `merge` defaults to in-place local file writes, like `git merge`.
- `--profile` means local authoring shape. Usually `pull --profile milvus` and `merge --profile milvus`, then `publish --profile zilliz`.
- `merge` supports `--target` fetch and `--remote` local snapshot input.
- `merge` never writes Feishu. Live tests may write Feishu only during setup.
- `merge` never writes publish receipts or base snapshots.
- Successful `publish --write` saves a local source snapshot as the next merge base.
- With base: line-based three-way merge.
- Without base: diff-region conflict, no automatic side selection.
- Conflict markers:

  ```md
  <<<<<<< LOCAL
  local content
  =======
  remote content
  >>>>>>> REMOTE
  ```

- Conflict exit code is `1`; clean/merged exit code is `0`.
- Existing conflict markers cause merge to fail before writing.
- Existing merge state causes merge to fail before writing.
- `merge --abort <file>` restores the pre-merge local file and removes merge state.
- V1 does not implement `--continue`, `--ours`, or `--theirs`.

export function baseSnapshotPath(input: { cwd: string; target: PublishReceiptTarget }): string {

  return join('.sync', 'feishu-md-sync', 'bases', `${input.target.kind}-${input.target.token}-local.md`);

export async function writeLocalBaseSnapshot(input: {
  cwd: string;
  target: PublishReceiptTarget;
  markdown: string;

  await mkdir(dirname(path), { recursive: true });
  await writeFile(path, input.markdown, 'utf8');

    path: baseSnapshotRelativePath({ target: input.target }),
    hash: hashText(input.markdown)
  };
}

export async function readLocalBaseSnapshot(input: {
  cwd: string;
  snapshot: LocalBaseSnapshot;
}): Promise<string | undefined> {
  try {
    return await readFile(join(input.cwd, input.snapshot.path), 'utf8');
  } catch (error) {
    if ((error as NodeJS.ErrnoException).code === 'ENOENT') return undefined;
    throw error;
  }
}

function splitLines(markdown: string): string[] {
  if (markdown.length === 0) return [];
  const parts = markdown.split('\n');
  return parts.slice(0, -1).map((line) => `${line}\n`).concat(parts.at(-1) === '' ? [] : [parts.at(-1)!]);
}

function joinLines(lines: string[]): string {
  return lines.join('');
}

function conflictLines(local: string[], remote: string[]): string[] {
  return [
    '<<<<<<< LOCAL\n',
    ...local,
    local.length > 0 && !local.at(-1)!.endsWith('\n') ? '\n' : '',
    '=======\n',
    ...remote,
    remote.length > 0 && !remote.at(-1)!.endsWith('\n') ? '\n' : '',
    '>>>>>>> REMOTE\n'
  ].filter((line) => line !== '');
}

- [ ] **Step 4: Implement LCS-based change extraction**

In `line-merge.ts`, add:

```ts
function lcsTable(a: string[], b: string[]): number[][] {
  const table = Array.from({ length: a.length + 1 }, () => Array<number>(b.length + 1).fill(0));
  for (let i = a.length - 1; i >= 0; i -= 1) {
    for (let j = b.length - 1; j >= 0; j -= 1) {
      table[i][j] = a[i] === b[j] ? table[i + 1][j + 1] + 1 : Math.max(table[i + 1][j], table[i][j + 1]);
    }
  }
  return table;
}

function changesFromBase(base: string[], next: string[]): LineChange[] {
  const table = lcsTable(base, next);
  const changes: LineChange[] = [];
  let i = 0;
  let j = 0;
  let pendingStart: number | undefined;
  let pendingEnd = 0;
  let pendingLines: string[] = [];

  const flush = () => {
    if (pendingStart === undefined) return;
    changes.push({ baseStart: pendingStart, baseEnd: pendingEnd, lines: pendingLines });
    pendingStart = undefined;
    pendingEnd = 0;
    pendingLines = [];
  };

  while (i < base.length || j < next.length) {
    if (i < base.length && j < next.length && base[i] === next[j]) {
      flush();
      i += 1;
      j += 1;
      continue;
    }

    if (pendingStart === undefined) pendingStart = i;

    if (j < next.length && (i === base.length || table[i][j + 1] >= table[i + 1]?.[j])) {
      pendingLines.push(next[j]);
      j += 1;
    } else {
      i += 1;
      pendingEnd = i;
    }
  }

  flush();
  return changes;
}
```

- [ ] **Step 5: Implement three-way merge**

Add:

```ts
export function mergeLines(input: { base: string; local: string; remote: string }): LineMergeResult {
  if (input.local === input.remote) {
    return {
      markdown: input.local,
      state: input.local === input.base ? 'clean' : 'merged',
      conflicts: 0,
      changed: input.local !== input.base
    };
  }

  const base = splitLines(input.base);
  const local = splitLines(input.local);
  const remote = splitLines(input.remote);
  const localChanges = changesFromBase(base, local);
  const remoteChanges = changesFromBase(base, remote);
  const output: string[] = [];
  let baseIndex = 0;
  let localIndex = 0;
  let remoteIndex = 0;
  let conflicts = 0;

  while (baseIndex < base.length || localIndex < localChanges.length || remoteIndex < remoteChanges.length) {
    const localChange = localChanges[localIndex];
    const remoteChange = remoteChanges[remoteIndex];
    const nextStart = Math.min(
      localChange?.baseStart ?? Number.POSITIVE_INFINITY,
      remoteChange?.baseStart ?? Number.POSITIVE_INFINITY,
      base.length
    );

    output.push(...base.slice(baseIndex, nextStart));
    baseIndex = nextStart;

    if (!localChange && !remoteChange) break;
    if (localChange && (!remoteChange || localChange.baseEnd <= remoteChange.baseStart)) {
      output.push(...localChange.lines);
      baseIndex = localChange.baseEnd;
      localIndex += 1;
      continue;
    }
    if (remoteChange && (!localChange || remoteChange.baseEnd <= localChange.baseStart)) {
      output.push(...remoteChange.lines);
      baseIndex = remoteChange.baseEnd;
      remoteIndex += 1;
      continue;
    }

    const localOverlap = localChange!;
    const remoteOverlap = remoteChange!;
    if (joinLines(localOverlap.lines) === joinLines(remoteOverlap.lines)) {
      output.push(...localOverlap.lines);
    } else {
      output.push(...conflictLines(localOverlap.lines, remoteOverlap.lines));
      conflicts += 1;
    }
    baseIndex = Math.max(localOverlap.baseEnd, remoteOverlap.baseEnd);
    localIndex += 1;
    remoteIndex += 1;
  }

  const markdown = joinLines(output);
  return {
    markdown,
    state: conflicts > 0 ? 'conflict' : markdown === input.local ? 'clean' : 'merged',
    conflicts,
    changed: markdown !== input.local
  };
}
```

export function mergeWithoutBase(input: { local: string; remote: string }): LineMergeResult {
  if (input.local === input.remote) {
    return {
      markdown: input.local,
      state: 'clean',
      conflicts: 0,
      changed: false
    };
  }

  const local = splitLines(input.local);
  const remote = splitLines(input.remote);
  const table = lcsTable(local, remote);
  const output: string[] = [];
  let i = 0;
  let j = 0;
  let conflicts = 0;

  while (i < local.length || j < remote.length) {
    if (i < local.length && j < remote.length && local[i] === remote[j]) {
      output.push(local[i]);
      i += 1;
      j += 1;
      continue;
    }

    const localChunk: string[] = [];
    const remoteChunk: string[] = [];
    while (i < local.length || j < remote.length) {
      if (i < local.length && j < remote.length && local[i] === remote[j]) break;
      if (j < remote.length && (i === local.length || table[i][j + 1] >= table[i + 1]?.[j])) {
        remoteChunk.push(remote[j]);
        j += 1;
      } else if (i < local.length) {
        localChunk.push(local[i]);
        i += 1;
      }
    }
    output.push(...conflictLines(localChunk, remoteChunk));
    conflicts += 1;
  }

  return {
    markdown: joinLines(output),
    state: 'conflict',
    conflicts,
    changed: true
  };
}

import { createHash } from 'node:crypto';
import { mkdir, readFile, rm, writeFile } from 'node:fs/promises';
import { dirname, join, relative } from 'node:path';
import type { PublishProfileName } from '../profiles/publish-profile.js';
import type { PublishReceiptTarget } from '../receipts/publish-receipt.js';

export type MergeStateFile = {
  version: 1;
  filePath: string;
  target?: PublishReceiptTarget;
  profile: PublishProfileName;
  startedAt: string;
};

export function mergeStateDir(input: { cwd: string; filePath: string }): string {
  const key = createHash('sha256').update(input.filePath).digest('hex').slice(0, 24);
  return join(input.cwd, '.sync', 'feishu-md-sync', 'merge-state', key);
}

export async function assertNoMergeState(input: { cwd: string; filePath: string }): Promise<void> {
  try {
    await readFile(join(mergeStateDir(input), 'state.json'), 'utf8');
  } catch (error) {
    if ((error as NodeJS.ErrnoException).code === 'ENOENT') return;
    throw error;
  }
  throw new Error('A merge is already in progress for this file. Resolve it or run merge --abort before starting another merge.');
}

export async function writeMergeState(input: {
  cwd: string;
  filePath: string;
  originalMarkdown: string;
  target?: PublishReceiptTarget;
  profile: PublishProfileName;
}): Promise<{ statePath: string }> {
  await assertNoMergeState({ cwd: input.cwd, filePath: input.filePath });
  const dir = mergeStateDir({ cwd: input.cwd, filePath: input.filePath });
  await mkdir(dir, { recursive: true });
  await writeFile(join(dir, 'original.md'), input.originalMarkdown, 'utf8');
  const state: MergeStateFile = {
    version: 1,
    filePath: relative(input.cwd, input.filePath),
    target: input.target,
    profile: input.profile,
    startedAt: new Date().toISOString()
  };
  const statePath = join(dir, 'state.json');
  await writeFile(statePath, `${JSON.stringify(state, null, 2)}\n`, 'utf8');

  cwd: string;
  filePath: string;
  profile: PublishProfileName;
  mode: RunMergeMode;

  basePath?: string;

      state: 'aborted',

    };

async function resolveRemoteMarkdown(input: {
  target?: PublishReceiptTarget;
  remotePath?: string;
  saveRemotePath?: string;
  profile: PublishProfileName;
  adapter: FeishuAdapter;
}): Promise<{
  markdown: string;
  warnings: string[];
  summary: RunMergeResult['remote'];
}> {
  if (input.remotePath) {
    const markdown = await readFile(input.remotePath, 'utf8');
    return {
      markdown,
      warnings: [],
      summary: {
        source: 'remote-file',
        hash: hashText(markdown)
      }
    };
  }

  const fetched = await input.adapter.fetchDocMarkdown({ doc: input.target!.token });
  const transformed = applyPullTransformForProfile(fetched.markdown, input.profile);
  if (input.saveRemotePath) {
    await writeFile(input.saveRemotePath, transformed.markdown, 'utf8');
  }
  return {
    markdown: transformed.markdown,
    warnings: transformed.warnings,
    summary: {
      source: 'target',
      hash: hashText(transformed.markdown),
      savedPath: input.saveRemotePath,
      revision: fetched.revision
    }
  };
}

async function resolveBaseMarkdown(input: {
  cwd: string;
  target?: PublishReceiptTarget;
  basePath?: string;
}): Promise<{
  markdown?: string;
  summary: NonNullable<RunMergeResult['base']>;
}> {
  if (input.basePath) {
    const markdown = await readFile(input.basePath, 'utf8');
    return {
      markdown,
      summary: {
        source: 'explicit',
        hash: hashText(markdown)
      }
    };
  }

  if (input.target) {
    const receipt = await readPublishReceipt({ cwd: input.cwd, target: input.target });
    if (receipt?.localBaseSnapshot) {
      const markdown = await readLocalBaseSnapshot({ cwd: input.cwd, snapshot: receipt.localBaseSnapshot });
      if (markdown !== undefined) {
        return {
          markdown,
          summary: {
            source: 'receipt',
            hash: hashText(markdown)
          }
        };
      }
    }
  }

  return {
    markdown: undefined,
    summary: {
      source: 'none'
    }
  };
}

type MergeCommandOptions = {
  target?: string;
  remote?: string;
  base?: string;
  profile?: string;
  check?: boolean;
  dryRun?: boolean;
  abort?: boolean;
  saveRemote?: string;
  format?: string;
};

export function registerMergeCommand(program: Command): void {
  program
    .command('merge')
    .description('merge remote Feishu/Lark Markdown changes into a local Markdown file')
    .argument('<markdown-file>', 'local Markdown file')
    .option('--target <url-or-token>', 'Feishu/Lark docx or wiki URL/token to fetch before merging')
    .option('--remote <file>', 'local remote snapshot Markdown file')
    .option('--base <file>', 'explicit merge base Markdown file')
    .option('--profile <profile>', 'local authoring profile: milvus | zilliz | none')
    .option('--check', 'check whether merge would be clean without writing')
    .option('--dry-run', 'show merge result metadata without writing')
    .option('--abort', 'restore the local file from the previous merge state')
    .option('--save-remote <file>', 'save fetched remote snapshot when using --target')
    .option('--format <format>', 'output format: pretty | json', 'pretty')
    .action(async (markdownFile: string, opts: MergeCommandOptions) => {
      const cwd = process.cwd();
      const config = await loadSyncConfig({ cwd });
      const profile = resolvePublishProfile({ cliProfile: opts.profile, config });
      const target = opts.target ? parseFeishuTarget(opts.target) : undefined;
      const mode = resolveMergeMode(opts);
      const result = await runMerge({
        cwd,
        filePath: path.resolve(cwd, markdownFile),
        target,
        remotePath: opts.remote ? path.resolve(cwd, opts.remote) : undefined,
        basePath: opts.base ? path.resolve(cwd, opts.base) : undefined,
        saveRemotePath: opts.saveRemote ? path.resolve(cwd, opts.saveRemote) : undefined,
        profile,
        mode,
        adapter: new LarkCliAdapter()
      });

      printFormatted(result, opts.format);
      if (result.state === 'conflict') process.exitCode = 1;
    });
}

function resolveMergeMode(opts: MergeCommandOptions): RunMergeMode {
  const selected = [opts.check, opts.dryRun, opts.abort].filter(Boolean).length;
  if (selected > 1) throw new Error('Choose only one of --check, --dry-run, or --abort.');
  if (opts.abort) return 'abort';
  if (opts.check) return 'check';
  if (opts.dryRun) return 'dry-run';
  return 'write';
}

Merges remote Markdown changes into the local source file. `merge` writes only local files. It never writes Feishu or publish receipts.

Options:

- `--target <url-or-token>`: fetch remote Markdown through `lark-cli docs +fetch`.
- `--remote <file>`: use an existing pulled remote snapshot.
- `--base <file>`: explicit three-way merge base.
- `--profile <milvus|zilliz|none>`: local authoring shape used to filter remote include/exclude tags.
- `--check`: report whether merge would be clean, merged, or conflicted without writing.
- `--dry-run`: print merge metadata without writing.
- `--abort`: restore the local file from the previous merge state.
- `--save-remote <file>`: save fetched remote snapshot when using `--target`.

Conflict exits with code `1`. Clean or auto-merged results exit `0`.
```
