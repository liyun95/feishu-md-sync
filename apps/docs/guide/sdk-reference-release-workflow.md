# SDK Reference Release

Use this page as a stable URL for releasing audited SDK reference docs to `web-content`. The canonical release sequence comes from:

```bash
md2feishu workflow show sdk-reference-web-content-release
```

- Workflow index: [Workflows](/guide/workflows)
- Safety gates: [Safety Gates](/reference/safety-gates)
- Command flags: [Commands](/reference/commands)
- Feishu authoring: [SDK Reference Workflow](/guide/sdk-reference-workflow)

Only start this workflow after a human explicitly decides that the audited Feishu reference draft is ready to publish to the docs website.
