# Section Sync

Use this page as a stable URL for the reviewed section sync workflow. The canonical step sequence comes from:

```bash
md2feishu workflow show reviewed-section-sync
```

- Workflow index: [Workflows](/guide/workflows)
- Safety gates: [Safety Gates](/reference/safety-gates)
- Command flags: [Commands](/reference/commands)

Section sync is for replacing one unique heading section while preserving Feishu content outside that section.
