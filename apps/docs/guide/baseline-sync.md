# First Baseline Sync

Use this page as a stable URL for the baseline sync workflow. The canonical step sequence comes from:

```bash
md2feishu workflow show baseline-sync
```

- Workflow index: [Workflows](/guide/workflows)
- Safety gates: [Safety Gates](/reference/safety-gates)
- Command flags: [Commands](/reference/commands)

Baseline sync is the workflow for pulling current Feishu content into local Markdown before editing or comparing changes.
