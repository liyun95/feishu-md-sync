# SDK Reference Workflow

Use this page as a stable URL for SDK reference authoring. The canonical Feishu writing and audit sequence comes from:

```bash
md2feishu workflow show sdk-reference-authoring
```

- Workflow index: [Workflows](/guide/workflows)
- Safety gates: [Safety Gates](/reference/safety-gates)
- Command flags: [Commands](/reference/commands)
- Release to web-content: [SDK Reference Release](/guide/sdk-reference-release-workflow)

This workflow ends after Feishu reference changes are written and audited. Exporting audited content into `web-content` is a separate, human-triggered release workflow.
